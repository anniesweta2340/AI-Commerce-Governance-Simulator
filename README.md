# AI Commerce Governance Simulator

A stylized model of the tradeoffs between monetization, transparency, and trust in AI-mediated commerce. Adjust three parameters — monetization level, disclosure type, and synthetic engagement rate — to observe effects on user trust, platform revenue, and brand ROI.

Built as a single React component (`governance-simulator.jsx`).

## Inputs

- **Monetization Level** — density of paid placements (low / medium / high)
- **Disclosure Type** — how sponsorship is communicated (none / sponsored label / full explanation)
- **Synthetic Engagement Rate** — share of interactions from non-human agents (0–50%)

## Outputs

- **Trust Score** (0–100)
- **Platform Revenue Index** (0–100)
- **Brand ROI Index** (0–100)
- A short interpretation of the resulting posture

## Note

Coefficients are illustrative, not derived from empirical data. The model is intended to make the structure of the tradeoff visible.
